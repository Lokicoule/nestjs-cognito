# NestJS Cognito

AWS Cognito authentication for NestJS applications.

<Note>
- **5-Minute Setup**: Get authentication running quickly
- **Production-Ready**: Built on [aws-jwt-verify](https://github.com/awslabs/aws-jwt-verify)
- **Type-Safe**: Full TypeScript support
</Note>

## Why NestJS-Cognito?

AWS Cognito integration can be complex: OIDC flows, JWT verification, token handling, AWS SDK configuration. NestJS-Cognito handles this for you.

### Key Features

**Simple Authentication**
```typescript
@Controller('profile')
export class ProfileController {
  @Get()
  @Authentication()  // âœ… Protected route
  getProfile(@CognitoUser() user: CognitoJwtPayload) {
    return user;
  }
}
```

**Authorization with Cognito Groups**
```typescript
@Get('admin')
@Authorization({ requiredGroups: ['admin'] })
getAdminData() {
  return { sensitive: 'data' };
}
```

**Flexible Token Extraction**
- Bearer tokens (default)
- Cookies
- Custom extractors
- Multiple sources

**Built-in Security**
- Automatic JWT verification
- Token expiration handling
- Issuer validation
- JWKS key management
